# Child Homework Tracker Bot

Telegram-бот для родителей, который позволяет **получать домашние задания детей в виде PDF-файлов** по номеру телефона.

---

## Функции

- **Регистрация по номеру телефона** (кнопка "Поделиться номером")
- **Автоматическое определение детей** из `parents.json`
- **Выбор ребёнка** по регистрационному номеру
- **Отправка PDF-файла** с домашним заданием
- **Поддержка разных форматов номера**: `+7`, `8`, без `+`
- **Работает в режиме Long Polling** (без вебхуков)
- **Автообновление списка родителей** при изменении `parents.json`
